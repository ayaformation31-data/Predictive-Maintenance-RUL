# Predictive Maintenance - Remaining Useful Life (RUL)

##  Project Overview

This project aims to predict the Remaining Useful Life (RUL) of aircraft engines using sensor data.

##  Objective

Predict the number of cycles remaining before engine failure using machine learning models.

##  Dataset

* Source: NASA Turbofan Engine Dataset (FD001)
* 100 engines with multiple time cycles
* 21 sensor measurements

##  Methods

* Data preprocessing and cleaning
* Feature selection
* Machine Learning models:

  * Linear Regression
  * Random Forest

##  Results

* Linear Regression:

  * RMSE: 37
  * MAE: 30

* Random Forest:

  * RMSE: 36.9
  * MAE: 26.4

##  Visualization

Predictions vs actual values show strong correlation.
